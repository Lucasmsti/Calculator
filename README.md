# Calculadora Simples

Uma calculadora simples construída com HTML, CSS e JavaScript.

## Como Usar

Basta abrir o arquivo `index.html` em seu navegador da web para usar a calculadora.

## Funcionalidades

- Adição (+)
- Subtração (-)
- Multiplicação (*)
- Divisão (/)
- Cálculos com números decimais
- Limpeza da operação atual
